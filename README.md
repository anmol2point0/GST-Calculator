## Features

* **Real-Time GST Engine:** Type standard math equations (e.g., `100 + 50`) and instantly apply or extract custom GST rates (0%, 6%, 9%, 12%, 18%, 28%) with zero lag.
* **Live Currency Converter:** Fetches real-time exchange rates from the European Central Bank. Includes an intelligent fallback to offline rates if you lose internet connection, complete with a live status badge.
* **Smart Price Book:** Save your frequently used items. Instantly search, filter, and send product prices directly into the main calculator with a single tap.
* **Multi-User Sessions:** Juggle up to three different calculations at once using the color-coded profile tabs.

## 🎨 The Cyberpunk Update (v1.1.0)

* **Neon Theme Engine:** Switch between Light, Dark, and the brand-new pure OLED "Neon" mode. Neon mode features glowing wireframe interfaces, pure black backgrounds, and a customizable electric color palette.
