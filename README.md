# 🧮 GST Calc Pro (v1.1.0)

A professional, offline-first financial utility app built with Flutter. GST Calc Pro combines a lightning-fast dynamic tax calculator, a live global currency converter, and a personal price book into one seamless, sensory-rich experience.

[![Download Stable](https://img.shields.io/badge/Download_Latest-v1.1.0-0A66C2?style=for-the-badge&logo=android)](https://github.com/anmol2point0/GST-Calculator/releases/latest)

---

## ✨ Core Features

* **Real-Time GST Engine:** Type standard math equations (e.g., `100 + 50`) and instantly apply or extract custom GST rates (0%, 6%, 9%, 12%, 18%, 28%) with zero lag.
* **Live Currency Converter:** Fetches real-time exchange rates from the European Central Bank (via Frankfurter API). Includes an intelligent fallback to offline rates if you lose internet connection, complete with a live status badge.
* **Smart Price Book:** Save your frequently used items. Instantly search, filter, and send product prices directly into the main calculator with a single tap.
* **Multi-User Sessions:** Juggle up to three different calculations at once using the color-coded profile tabs.

## 🎨 The Cyberpunk Update (New in v1.1.0)

* **Neon Theme Engine:** Switch between Light, Dark, and the brand-new pure OLED "Neon" mode. Neon mode features glowing wireframe interfaces, pure black backgrounds, and a customizable electric color palette.
* **Advanced Haptics:** The app now physically responds to your touch. Enjoy crisp clicks for numbers, heavy thuds for deletions, and special "glitch" haptic sequences exclusive to Neon mode.
* **Rock-Solid Stability:** Fully optimized memory management with zero background leaks, infinite-loop safeguards, and dynamic system-loading to prevent screen flashes.

## 🛠️ Built With
* **Framework:** Flutter & Dart
* **Architecture:** Offline-first, dynamic state management
* **Key Packages:** `math_expressions`, `shared_preferences`, `http`, `url_launcher`

## 📱 Installation
1. Go to the [Releases Page](https://github.com/anmol2point0/GST-Calculator/releases/latest).
2. Download the latest `GST-Calc-Pro.apk` file to your Android device.
3. Open the file and click **Install**. 
*(Note: Android may ask you to "Allow installation from unknown sources" since it is not from the Play Store).*
